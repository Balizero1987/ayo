# Assets Directory

Questa cartella contiene file binari e asset che non devono essere committati nel repository Git.

## Struttura

- `images/` - Immagini e screenshot
- `debug/` - File di debug temporanei

## Note

- Tutti i file in questa directory sono ignorati da `.gitignore`
- I file qui contenuti sono solo per riferimento locale
- Non committare file grandi o temporanei qui
