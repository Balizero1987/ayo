**Import barang pribadi (personal effects) punya rules khusus** dari Bea Cukai.

**Kategori Import Barang Pribadi:**

**1. Barang Bawaan Penumpang (Hand-Carry):**
- **De minimis threshold:** **USD 500 per orang** (value barang)
- **Jika nilai < USD 500:** **Bebas bea masuk** (gratis)
- **Jika nilai > USD 500:** Kena bea masuk + PPN 11%

**Contoh:**
- Anda bawa laptop (USD 1,000) + pakaian (USD 200) = **Total USD 1,200**
- **Free allowance:** USD 500
- **Taxable value:** USD 1,200 - USD 500 = **USD 700**
- **Duty (assume 10%):** USD 700 √ó 10% = **USD 70**
- **PPN 11%:** (USD 700 + USD 70) √ó 11% = **USD 84.7**
- **Total bayar:** USD 70 + USD 84.7 = **USD 154.7** (sekitar IDR 2.4 juta)

---

**2. Barang Pindahan (Relocation Goods via Cargo):**
- **Untuk expat yang pindah ke Indonesia** (ada KITAS/work permit)
- **Bebas bea masuk** jika:
  - ‚úÖ Anda punya **KITAS** (E26, E28A, E33G, dll) yang valid
  - ‚úÖ Barang adalah **used personal effects** (sudah dipakai, bukan baru)
  - ‚úÖ **Quantity reasonable** (tidak untuk dijual)
  - ‚úÖ Arrive **within 6 months** dari KITAS issued

**Syarat dokumen:**
- Copy KITAS
- **Surat pernyataan** bahwa barang adalah milik pribadi (not for sale)
- **Inventory list** (daftar barang dengan estimated value)
- **Invoice/proof of ownership** (receipt, photo)

**Timeline:**
- Shipment arrive di pelabuhan: **Day 1**
- Customs clearance (jika lancar): **3-7 hari kerja**
- **Total:** 1-2 minggu dari arrival sampai barang delivered ke rumah

**3. Barang Komersial (Commercial Import):**
- **Untuk bisnis** (import barang untuk dijual atau produksi)
- **Wajib bayar:**
  - **Bea Masuk (Import Duty):** 0-150% (depends on HS Code)
  - **PPN (VAT):** 11%
  - **PPh 22 (Import Income Tax):** 2.5-10%

**Contoh:**
- Import furniture untuk dijual (HS Code 9403 ‚Äî furniture)
- **CIF value:** USD 10,000 (Cost + Insurance + Freight)
- **Duty rate:** 20% (furniture)
- **Calculation:**
  - Bea Masuk: USD 10,000 √ó 20% = **USD 2,000**
  - **Nilai Dasar PPN:** USD 10,000 + USD 2,000 = **USD 12,000**
  - PPN 11%: USD 12,000 √ó 11% = **USD 1,320**
  - PPh 22 (2.5%): USD 12,000 √ó 2.5% = **USD 300**
  - **Total tax:** USD 2,000 + USD 1,320 + USD 300 = **USD 3,620** (sekitar IDR 56 juta)

**Key Terms Explained:**

**CIF (Cost, Insurance, Freight):**
- **Meaning:** Total value **includes** cost barang + insurance + freight sampai pelabuhan Indonesia
- **Example:** Furniture cost AUD 5,000 + freight AUD 1,500 + insurance AUD 100 = **CIF AUD 6,600**
- **Bea Cukai uses CIF value** untuk calculate duty

**FOB (Free On Board):**
- **Meaning:** Value **hanya cost barang**, tidak termasuk freight/insurance
- **Example:** FOB AUD 5,000 (freight + insurance dibayar terpisah)
- **Note:** Jika declare FOB, Bea Cukai akan **add estimated freight + insurance** untuk calculate duty

**Customs Valuation:**
- **Rule:** Bea Cukai pakai **CIF value** (total landed cost)
- **Jika invoice FOB:** Bea Cukai akan estimate freight + insurance ‚Üí convert ke CIF
- **Recommendation:** Declare as **CIF** dari awal (simpler, no dispute)

**Consultant:**
**HS Code (Harmonized System Code):**

**Apa itu HS Code:**
- **HS Code** adalah **international classification system** untuk goods (dipakai di 200+ negara)
- **Format:** 6-10 digit angka (e.g., **9403.60.00** = Wooden furniture)
- **Function:** Determine **duty rate** untuk setiap kategori barang
- **Maintained by:** World Customs Organization (WCO)

**Example HS Codes:**

| Barang | HS Code | Description | Duty Rate (Indonesia) |
|--------|---------|-------------|----------------------|
| **Laptop** | 8471.30 | Portable computers | 0% (electronics free duty) |
| **Sofa (wooden)** | 9403.60 | Wooden furniture | 20% |
| **Clothing (used)** | 6309.00 | Used clothing | 25% |
| **Dining table** | 9403.60 | Wooden furniture | 20% |
| **Books** | 4901.99 | Printed books | 0% |
| **Bicycle** | 8712.00 | Bicycles | 15% |
| **Wine (alcohol)** | 2204.21 | Wine in bottles | 150% + excise |
| **Mattress** | 9404.21 | Mattresses | 20% |

**How Bea Cukai Determines Duty Rate:**

**Step 1: Classification (HS Code):**
- Customs officer (or broker) **classifies** barang berdasarkan HS Code
- **Reference:** Indonesia Customs Tariff Book (BTKI ‚Äî Buku Tarif Kepabeanan Indonesia)
- **Dispute:** Jika Anda disagree dengan classification, bisa **appeal** (tapi jarang menang)

**Step 2: Valuation (CIF Value):**
- Officer checks **CIF value** dari invoice
- **Benchmark:** Compare dengan **reference price** (Bea Cukai punya database harga pasaran)
- **Red flag:** Jika declared value **too low** (e.g., sofa declared USD 50 tapi market price USD 500) ‚Üí officer bisa **reject** dan minta re-declare

**Step 3: Origin (Country of Origin):**
- Check **Certificate of Origin** (if applicable)
- **Preferential rate:** Jika barang dari **FTA partner country** (e.g., Australia, ASEAN, China) ‚Üí bisa dapat **discounted duty**
- **Example:** Furniture dari Australia under **IA-CEPA** ‚Üí duty bisa **0% instead of 20%**

**Step 4: Calculation:**
- **Formula:** Duty = CIF Value √ó Duty Rate
- **Then add PPN:** PPN = (CIF + Duty) √ó 11%
- **Then add PPh 22 (if commercial):** PPh 22 = (CIF + Duty) √ó 2.5-10%

**Red Lane vs Green Lane:**

**Green Lane (Jalur Hijau):**
- **Low-risk shipment** (personal effects, low value)
- **No physical inspection** ‚Äî automatic clearance
- **Timeline:** 1-2 hari
- **Probability:** ~70% untuk Barang Pindahan with proper docs

**Red Lane (Jalur Merah):**
- **High-risk shipment** (commercial goods, high value, suspicious declaration)
- **Physical inspection required** ‚Äî officer opens boxes, check items
- **Timeline:** 3-7 hari (waiting for inspection slot)
- **Probability:** ~30% (random selection or flagged by system)

**Yellow Lane (Jalur Kuning):**
- **Document verification** ‚Äî officer checks docs tapi tidak buka boxes
- **Timeline:** 2-4 hari

**Tips to Avoid Red Lane:**
- ‚úÖ **Accurate declaration:** Don't undervalue goods (honesty is best)
- ‚úÖ **Complete documents:** All docs ready (invoice, packing list, KITAS)
- ‚úÖ **Clear description:** Don't write "miscellaneous items" ‚Äî be specific
- ‚úÖ **Photos:** Attach photos of items (show they're used, personal)

**What happens in Red Lane:**
- Officer schedules **physical inspection** (1-3 hari waiting)
- You (or forwarder) must be **present** during inspection
- Officer opens boxes, **verifies items** vs packing list
- If mismatch (e.g., you declared "10 shirts" tapi found "50 shirts for resale") ‚Üí **penalty**

**Types of Penalties (Bea Cukai):**

**1. Administrative Fine (Denda Administrasi):**
- **Untuk:** Late document submission, minor errors
- **Amount:** IDR 1-5 juta (fixed)

**2. Underpayment Penalty:**
- **Untuk:** Undervaluation (declare lower value than actual)
- **Formula:** **(Correct Duty - Paid Duty) √ó 2** (double the difference)
- **Example:**
  - Correct duty: USD 1,000
  - You declared value low ‚Üí paid duty: USD 500
  - Penalty: (USD 1,000 - USD 500) √ó 2 = **USD 1,000**
  - **Total bayar:** USD 1,000 (correct duty) + USD 1,000 (penalty) = **USD 2,000**

**3. Smuggling Penalty (Severe):**
- **Untuk:** Intentional misdeclaration, concealment, prohibited goods
- **Penalty:** **100-1000%** dari duty value + possible **criminal charges**
