# Percakapan Visa B1 — Bahasa Bali Halus

**Client:**  
Puntên, titiang nunas iraga jelasinangang. Visa B1 punika dadosangang napi indik punapi?

**Konsultan:**  
Visa B1 punika visa kunjungan sane dados kaanggén meeting bisnis, survei usaha, negosiasi, utawi aktivitas non-kerja. Dados panjenengan prasida rauh ring Indonesia tanpa nyantosang upah utawi kerja.

**Client:**  
Oh, dados yen titiang wenten rencana nyari informasi investasi lan ngider lokasi, punika dados nganggé Visa B1?

**Konsultan:**  
Dados pisan. Visa B1 punika pas pisan kangge survei pasar, ketemu notaris, nyidangang lokasi usaha, utawi riset investasi.

**Client:**  
Yen titiang nenten presentasi ide lan ketemu calon partner, dados?

**Konsultan:**  
Dados, asal tan wenten kontrak kerja sareng tan nyantosang peparuman. Meeting, seminar, presentasi — masih kaidaang.

**Client:**  
Napi sukatin tinggalné?

**Konsultan:**  
Visa B1 ngicenang 60 rahina (2 sasi). Puniki single entry — yen medal Indonesia, visa langsung nenten berlaku.

**Client:**  
Syarat-syaraté napi?

**Konsultan:**  
– Foto paspor  
– Foto selfie  
– Alamat tinggal ring Indonesia  
– Bank statement 3 bulan terakhir  
– Itinerary meeting/survei

**Client:**  
Prosesé lami?

**Konsultan:**  
7–8 rahina kerja, sareng punika kedah kaurus offshore.

**Client:**  
Biayané napi?

**Konsultan:**  
Rp 3.500.000 kalebet pengurusan.

**Client:**  
Prasida diperpanjang?

**Konsultan:**  
Tan prasida. Yen pesan langkung sukatin, kudu ngaturang aplikasi anyar.

**Client:**  
Larangan khusus?

**Konsultan:**  
– Tan dados makarya  
– Tan dados nyantosang gaji  
– Tan ngelaksanayang promosi dagang  
– Tan ngajual barang/jasa  
– Tan laksana aktivitas sane nyantosang peparuman dané

**Client:**  
Suksma, titiang siapang dokumen-dokumenné.

**Konsultan:**  
Suksma malih. Titiang siap dampingin sakancan prosesné.
